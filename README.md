## How to Install

1. Download the latest version of RStudio for macOS.  
2. Open the DMG installer and move RStudio.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

## Overview

<a href="#">RStudio for macOS</a> is crafted for analysts who treat R as their primary language for statistics, modelling and data exploration. Scripts, consoles, plots and environment views are brought together so that each change in code can be reflected immediately in visuals and objects. Project-centric organisation helps keep dependencies, data and outputs contained, which is important when juggling multiple analyses across clients or research threads. This reduces the friction of context switching and keeps work reproducible.

Integrated support for packages, documentation and vignettes encourages users to stand on the shoulders of the broader R ecosystem. Data import wizards, tidyverse-friendly views and <a href="#">R Markdown or Quarto</a> documents streamline the process of turning analysis into reports, dashboards or presentations. Version control integration and terminal panes make it possible to keep R work closely aligned with the rest of a team’s tooling stack, whether in academia, industry or finance.

The macOS build of RStudio embraces the platform’s look and feel while ensuring that heavy plotting and modelling tasks remain responsive. External monitors can be dedicated to plots or viewers while the main editor stays on a laptop screen. Over time, the IDE becomes not just a code editor but a command centre for how data‑centric questions are asked, answered and communicated to stakeholders.

## Major Features
- Project-based organisation for R scripts and assets  
- Integrated console, editor, plots and environment views  
- Support for R Markdown, Quarto and reproducible reports  
- Data viewers tuned for tabular and tidy data structures  
- Package management and documentation access in-IDE  
- Version control integration for collaborative analysis  
- Notebook-style workflows for exploratory modelling  
- macOS-optimised layout for single and multi-screen setups  
- Tools for interactive apps and dashboards via Shiny  
- Helpful for teaching, research and production analytics  
- Seamless navigation between code, results and narrative  
- Designed to make R a first-class citizen on macOS
